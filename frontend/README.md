## Requisitos do Frontend (React + TailwindCSS)

### Interface Moderna
- [ ] Desenvolver uma UI limpa e responsiva utilizando TailwindCSS.  
- [ ] Garantir que o design transmita sobriedade e confiança, típicos do setor jurídico.

### Chat Interativo
- [ ] Criar uma interface de chat onde o histórico da conversa é preservado.

### Visualização de Fonte (Diferencial)
- [ ] Mostrar, nas respostas da IA, qual trecho do documento original foi utilizado (citação da fonte).

### Feedback de Estado
- [ ] Exibir indicadores de carregamento como: "Lendo documento..." e "Consultando jurisprudência..." para lidar com a latência das APIs de IA.
