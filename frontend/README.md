## Requisitos do Frontend (React + TailwindCSS)

- [ ] Interface Moderna: Desenvolver uma UI limpa e responsiva utilizando TailwindCSS. O design deve passar sobriedade e confiança, típicos do setor jurídico.
- [ ] Chat Interativo: Criar uma interface de chat onde o histórico da conversa é preservado.
- [ ] Visualização de Fonte (Diferencial): Quando a IA responder, mostrar qual trecho do documento original foi utilizado para basear a resposta (citação da fonte).
- [ ] Feedback de Estado: Mostrar indicadores de carregamento ("Lendo documento...", "Consultando jurisprudência...") para lidar com a latência das APIs de IA.
